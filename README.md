# Genomic Data Processing & Visualization in Python

A collection of Python scripts developed to parse, analyze, and visualize genomic and proteomic sequence data using standard scientific libraries (`matplotlib`, `numpy`).

## Key Features & Scripts
- * Biological Data Visualisation includes:

    **GC Content Analysis (`plot_gc_content.py`):** Calculates GC percentage across DNA sequences and dynamically color-codes values based on biological thresholds.
- * DNA and sequence analysis includes: 

-  **Amino Acid Composition Profiler (`plot_amino_acid_compositions.py`):** Generates grouped bar charts comparing the 20 standard amino acid distributions across multiple protein sequences.

- * Protein and Amino Acid Analysis: 
- **Sequence Length vs. GC Scatter Plot (`plot_length_gc.py`):** Implements Interquartile Range (IQR) statistical modeling to isolate sequence outliers and visualize multi-variable groupings.
-  **Plot GC content of DNA sequences** and plots the result as a bar chart using MatPlotLib. Highlight bars for sequences with GC conent above 60%.
